## Description

BWL is a web project designed for interacting with images shared from several channels of the KOMU NCC Discord. The code is built using NestJS + MongoDB, ReactJS, Webpack,...

## Installation

```bash
$ npm install
```

## Running the app

- development
```bash
$ npm run start
```

- watch mode
```bash
$ npm run start:dev
```

- production mode
```bash
$ npm run start:prod
```

## Test

- unit tests
```bash
$ npm run test
```

- e2e tests
```bash
$ npm run test:e2e
```

- test coverage
```bash
$ npm run test:cov
```
