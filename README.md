## Description

BWL is a web project designed for interacting with images shared from several channels of the KOMU NCC Discord. The code is built using NestJS + MongoDB, ReactJS, Webpack,...

## Getting Started

# Installation

```bash
$ npm install
```

# Edit
- If you've edited the code, run this command before executing the 'run' command.
```bash
$ npx webpack
```

# Run

- development
```bash
$ npm run start
```

- watch mode
```bash
$ npm run start:dev
```

- production mode
```bash
$ npm run start:prod
```

# Test

- unit tests
```bash
$ npm run test
```

- e2e tests
```bash
$ npm run test:e2e
```

- test coverage
```bash
$ npm run test:cov
```
